# ⚡ クイックスタート

このガイドでは、最速で仕事モンスター診断アプリを起動する手順を説明します。

## 🎯 目標

- 5分以内にアプリを起動

- 最初の診断を完了

## 📋 事前準備

以下がインストールされていることを確認：

```bash
node --version  # v18以上
npm --version   # v9以上
```

## 🚀 5ステップで起動

### Step 1: プロジェクトディレクトリに移動

```bash
cd /Users/kodaiseta/work-monster-quiz
```

### Step 2: 依存パッケージのインストール（初回のみ）

```bash
npm install
```

### Step 3: Anthropic APIキーを設定

1. [Anthropic Console](https://console.anthropic.com/) にアクセス
2. APIキーを取得（無料で$5分のクレジット付与）
3. `.env.local` ファイルを編集:

```bash
# .env.local
ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxxxxxxxxxxxxxxxx
```

**重要:** `your_api_key_here_replace_this` の部分を実際のAPIキーに置き換えてください！

### Step 4: 開発サーバーを起動

```bash
npm run dev
```

以下のメッセージが表示されれば成功：

```
✓ Ready in 1766ms
- Local:        http://localhost:3000
```

### Step 5: ブラウザでアクセス

http://localhost:3000 を開く

## 🐛 トラブルシューティング

### Q: AI機能がエラーになる

**症状:**
```
AIアドバイスの生成に失敗しました
```

**解決策:**
1. `.env.local` ファイルが存在するか確認
2. APIキーが正しくコピーされているか確認（余計なスペースなし）
3. 開発サーバーを再起動:
   ```bash
   # Ctrl+C で停止
   npm run dev  # 再起動
   ```

### Q: ページが真っ白

**解決策:**
1. ブラウザのコンソールを開く（F12キー）
2. エラーメッセージを確認
3. 開発サーバーのターミナルを確認

### Q: 3000番ポートが使用中

**解決策:**
```bash
# 別のポートで起動
PORT=3001 npm run dev
```

その後、http://localhost:3001 にアクセス

### Q: APIキーが無効

**症状:**
```
authentication_error
```

**解決策:**
1. [Anthropic Console](https://console.anthropic.com/) でAPIキーを確認
2. 新しいキーを発行
3. `.env.local` を更新
4. サーバーを再起動

## 💡 Tips

### 開発効率化

**コードの自動リロード**
- ファイルを編集すると自動でブラウザが更新されます
- 環境変数（.env.local）の変更時はサーバーの再起動が必要

**複数ブラウザでテスト**
- http://localhost:3000 は同じネットワーク上の他のデバイスからもアクセス可能
- スマホでの表示確認も簡単

**エラーの確認**
- ブラウザのコンソール（F12）
- ターミナルのサーバーログ
- 両方を同時に確認するのがベスト

### AI機能の最適化

**開発中のコスト節約**
- テスト時は同じ回答パターンで診断（キャッシュされる）
- 頻繁にテストする場合は一時的にAI機能をコメントアウト

**本番環境**
- Vercelにデプロイ後も同様に動作します
- 環境変数の設定を忘れずに
